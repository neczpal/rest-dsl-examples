# REST DSL Examples

This repository contains example projects demonstrating the use of the [REST DSL](https://github.com/neczpal/rest-dsl) language.

## Examples

### Petstore Example

A classic Petstore example that shows how REST DSL can be used to generate a functional server and a client application.

- **DSL Definition**: The API is defined in `petstore/petstore.rdsl`.
- **Server**: A generated Spring Boot server project is located in `petstore/server/`.
- **Client**: A generated Java client application project is located in `petstore/client/`.
